# 既存のCommerce コードの読み込み

Adobeでは、空のテンプレートから、または既存のコードを読み込むことで、Commerce on cloud infrastructure プロジェクトを作成することをお勧めします。 まず空白のテンプレートから始め、次に既存のコードを新しいプロジェクトに読み込むことをお勧めします。 これにより、プロジェクトをサポートするために必要なファイルがすべて揃っています。

既存のコードを読み込むための完全なワークフローには、次の手順が含まれます。

- テンプレートコードの置き換え
- データベースとメディアコンテンツの読み込み
- キャッシュをクリアしてインポートを検証
