# Cloud Docker for Commerce

Cloud Docker for Commerceは、Cloud Tools Suiteの一部です。 Cloud Docker ツールは、Adobe Commerce オンプレミスまたはクラウドインフラストラクチャプロジェクトをローカルのクラウドのような環境で開発およびテストするために使用できます。

- 最新のパッケージアップデートについては、[Cloud Docker パッケージリリースノート &#x200B;](../release-notes/cloud-docker.md)を参照してください。
- _Commerce開発者向けドキュメント_ サイトの[Cloud Docker for Commerce ガイド &#x200B;](https://developer.adobe.com/commerce/cloud-tools/docker/)。

## ローカル開発

Docker環境は、便利なローカル開発とテストのために、Adobe Commerce on cloud infrastructure `integration`環境をエミュレートします。 3つの重要な要素が必要です。

1. [Adobe Commerce v2 クラウドプロジェクトファイル](../project/file-structure.md)
1. [Docker Compose](https://www.docker.com/get-started/)
1. [ECE-Tools パッケージ](install-package.md)

Docker開発環境の操作を開始するには、前提条件とインストールについては、_Cloud Docker for Commerce ガイド_&#x200B;の[ローカル開発](https://developer.adobe.com/commerce/cloud-tools/docker/setup/)を参照してください。
