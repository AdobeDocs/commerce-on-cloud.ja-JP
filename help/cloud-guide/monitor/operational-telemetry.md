# 運用用遠隔測定サービス

Operational Telemetry サービスは、web サイトまたはアプリケーション上のクライアントサイドのトラフィックをリアルタイムで監視します。 このサービスは、ユーザー自身ではなく、web サイトのエンゲージメントを監視することで、パフォーマンスを最適化する鍵となる指標とデータを収集することに重点を置いています。 Operational Telemetryを使用すると、リクエストがブラウザーに返されるまで、URLの開始から直接キーパフォーマンス指標が追跡されます。

## 実装の詳細

このサービスは現在、一部のAdobe Commerce サイトにロールアウトされています。

## アクセスと管理

Operational Telemetry データにアクセスするか、サービスを無効にするお客様は、Adobe サポートにお問い合わせください。

>[!NOTE]
>
>サービスの詳細については、[運用上のテレメトリ &#x200B;](https://www.aem.live/docs/operational-telemetry)を参照してください。
